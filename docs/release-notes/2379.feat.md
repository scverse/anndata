Create a new setting {attr}`anndata.settings.restrict_index_types` to allow for custom in-memory indexes (i.e., for {attr}`~anndata.AnnData.obs` and {attr}`~anndata.AnnData.var`) to be passed to an {class}`~anndata.AnnData` objects without being converted to a string type.
Note that indexes recognized by {func}`pandas.api.types.is_integer_dtype` will still be stringified regardless of the setting.
