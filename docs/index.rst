:class:`~anndata.AnnData` provides a scalable way of keeping track
of data and learned annotations.

* Discuss development on `GitHub <https://github.com/theislab/anndata>`_.
* See `Scanpy's documentation <https://scanpy.readthedocs.io/>`__ for usage and discussions related to single cell data.
* anndata was initially built for Scanpy `(Genome Biology, 2018) <https://doi.org/10.1186/s13059-017-1382-0>`__.
